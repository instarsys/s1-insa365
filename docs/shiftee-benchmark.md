## 1. 출퇴근기록 — 달력형 (`/attendances/calendar`)

### 레이아웃
- **매트릭스 테이블**: 행=직원, 열=1~말일
- **헤더 버튼**: 다운로드 | 업로드 | + 출퇴근기록 추가하기

### 필터 (6종)
| 필터 | 유형 | 설명 |
|------|------|------|
| 월 선택 | DatePicker (월) | 년/월 탐색 |
| 지점 | Multi-select | 복수 지점 선택 |
| 활성/비활성 직원 | Toggle | 퇴사자 포함 여부 |
| 지각 표시 범위 | Range (0~60분) | 지각 기준 분 설정 |
| 조퇴 표시 범위 | Range | 조퇴 기준 분 설정 |
| 휴가 표시 | Checkbox | 셀에 휴가 정보 표시/숨김 |

### 셀 디자인
- 출근시간(위) / 퇴근시간(아래) 2줄 표시
- **지점별 색상 바**: 좌측 2px, 지점마다 고유 컬러 (금산본사=green, 남청주=cyan)
- **미퇴근 표시**: 빨간 점(●) 아이콘
- **합계 행**: 일별 출근 인원수
- **합계 열**: 직원별 출근 일수

### 페이지네이션
- 옵션: 100 / 200 / 300 / 400 / 500 / 1000 / 2000 (대용량 지원)

### 인터랙션
- 날짜 헤더 클릭 → 목록형 이동
- 직원 클릭 → 직원 상세
- 셀 클릭 → 수정 모달

- **페이지네이션**: 10 / 25 / 50 / 100 / 200 / 300 / 500 / 1000

## 5. 휴가 관리하기 모달

- **직원 선택**: 검색 + 고급 필터 + 그룹별(최고관리자/직원) + 모두 선택
- **월간 달력**: 공휴일(설날 연휴 등) 마킹, 날짜 선택으로 휴가 부여
- **직원 목록**: 이름 + 지점 표시

## MVP 적용 대상

시프티 분석 결과 중 현 MVP 스코프 내 즉시 적용 가능한 개선사항:

### 1. 달력형 — 미퇴근 빨간 점(●) 표시
- **Before**: 미퇴근 시 `-` 텍스트 표시
- **After**: 빨간 ● 아이콘으로 시각적 강조
- **근거**: 시프티 동일 패턴, 50명+ 환경에서 미퇴근자 즉시 식별 필수

### 2. 달력형 — 페이지네이션 옵션
- **Before**: 50명 고정
- **After**: 50/100/200/500 선택 가능
- **근거**: 시프티는 최대 2000까지 지원. SME 50~300명 타겟에 유연성 필요

### 3. 달력형 — 휴가 표시 토글
- **Before**: 휴가 셀 항상 표시
- **After**: 체크박스로 휴가 정보 표시/숨김 전환
- **근거**: 시프티 동일 패턴. 출퇴근 시간만 집중 확인할 때 유용

### 4. 목록형 — 페이지네이션 옵션
- **Before**: 25건 고정
- **After**: 10/25/50/100 선택 가능
- **근거**: 시프티 동일 패턴. 데이터 양에 따라 사용자가 조절

### 5. 목록형 — 요약 바 총 시간 추가
- **Before**: 총 건수 + 정규/연장/야간 시간만 표시
- **After**: 총 시간(h) 합산 추가
- **근거**: 시프티 요약 바에 "총 시간" 표시. 전체 근무 시간 한눈에 파악
